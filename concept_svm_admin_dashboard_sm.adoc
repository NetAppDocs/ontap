= Learn about the SVM administrator dashboard in ONTAP System Manager
:toclevels: 1
:hardbreaks:
:nofooter:
:icons: font
:linkattrs:
:imagesdir: ./media/

[.lead]
Beginning with ONTAP 9.19.1, ONTAP cluster administrators can configure a unique login for SVM administrators so they can monitor and manage their own SVMs. This is particularly useful in a tenanted ONTAP environment, where the cluster administrator can provide an SVM dashboard for each tenant. 

Cluster administrators can allow the SVM administrator the ability to create volumes. 

SVM administrators receive login credentials from their cluster administrator. In the System Manager Dashboard, SVM administrators can view the Storage and Volume pages. The Storage page provides information about the protocols, network interfaces, details, and performance metrics for the SVM. The Volume page provides information about the SVM volumes and, if allowed by the cluster administrator, can allow the SVM administrator to add new volumes.
