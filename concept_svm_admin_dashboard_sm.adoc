= Learn about the SVM administrator dashboard in ONTAP System Manager

:toclevels: 1
:hardbreaks:
:nofooter:
:icons: font
:linkattrs:
:imagesdir: ./media/

[.lead]
Beginning with ONTAP 9.19.1, SVM administrators can monitor and manage their SVMs using a dedicated System Manager dashboard. This is particularly useful in a tenanted ONTAP environment, where the cluster administrator can provide an SVM dashboard for each tenant.

Cluster administrators can allow the SVM administrator the ability to create volumes.

The cluster administrator provides login credentials to SVM administrators. In the System Manager Dashboard, SVM administrators can view SVM details and the Volumes page. The dashboard shows SVM protocols, network interfaces, details, and performance metrics. The Volume page provides information about the SVM volumes and, if allowed by the cluster administrator, the ability for the SVM administrator to add new volumes.
