= TCP port requirements for ONTAP SnapMirror active sync
:nofooter:
:icons: font
:linkattrs:
:imagesdir: ../media/

[.lead]
SnapMirror active sync and ONTAP Mediator require access to the following TCP ports to enable communication with ONTAP clusters.

== TCP ports required by SnapMirror

[cols=5,options="header"]
|===
| Port | Source | Direction | Destination | Purpose
| 3260 (TCP)
| Node intercluster
| Both
| Node intercluster
| iSCSI used for LUN consensus

| 11104/TCP and 11105/TCP
| Node intercluster
| Both
| Node intercluster
a| Intercluster communication sessions

SnapMirror data transfer

| 443/(TCP/HTTPS)
| Node intercluster
| Both
| Node intercluster
a| Management traffic (intercluster LIFs)

| 10000/TCP
| Node management/cluster LIFs
| Both
| Optional
a| Used for communication with Active IQ

|===

== TCP ports required by ONTAP Mediator

[cols=5,options="header"]
|===
| Port/Services | Source | Direction | Destination | Purpose
| 22/TCP
| Management host
| Inbound
| ONTAP Mediator
| SSH/ONTAP Mediator management

| 31784/TCP
| Cluster management
| Inbound
| ONTAP Mediator web server
a| REST API (HTTPS)

| 31784/TCP
| Node management LIFs
| Inbound
| ONTAP Mediator web server
a| ONTAP Mediator service (essential for Active Sync quorum)

| 3260/TCP
| Node management LIFs
| Inbound
| ONTAP Mediator iSCSI targets
a| iSCSI data connection for mailboxes

|===
