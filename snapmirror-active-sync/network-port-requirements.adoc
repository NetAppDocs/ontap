= Network port requirements
:icons: font
:linkattrs:
:imagesdir: ../media/

[.lead]
SnapMirror Active Sync has the following network port requirements. 

In most cases, SnapMirror Active Sync will be run on an internal network with no port restrictions. If IP traffic is restricted via firewalls or other mechanisms, the following port rules are required:

== Communication between nodes via the intercluster/peer links

[options="header"]
|===
| Port | Source | Direction | Destination | Purpose
| 3260/tcp| intercluster LIFs | both | intercluster LIFs | LUN consensus
a| 11104/tcp +
11105/tcp | intercluster LIFs | both | intercluster LIFs a| Intercluster communication sessions and SnapMirror data transfer
| 443/tcp (HTTPS) | intercluster LIFs | both | intercluster LIFs| Intercluster management traffic
| 10000/tcp a| cluster-mgmt LIFs + 
node-mgmt LIFs | both | optional | Communication with Active IQ Unified
|===

== Communication between the mediator and cluster nodes

[options="header"]
|===
| Port | Source | Direction | Destination | Purpose
| 22/tcp (SSH) | Management host | Inbound | ONTAP Mediator | ONTAP Mediator management
| 31784/tcp (HTTPS) | cluster-mgmt LIFs| Inbound | ONTAP Mediator | REST API 
| 31784/tcp | node-mgmt LIFs| Inbound | ONTAP Mediator a| ONTAP Mediator quorum service 
|===
