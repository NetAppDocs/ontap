= Monitor ONTAP volume migration
:icons: font
:imagesdir: ../media/


[.lead]
In addition to monitoring the overall SVM migration with the `vserver migrate show` command, you can monitor the migration status of the volumes the SVM contains.

.Steps

. On the destination cluster, check the volume migration status:
+
[source,cli]
----
vserver migrate show-volume
----
