= Manually remove source ONTAP SVM after client cutover
:icons: font
:imagesdir: ../media/


[.lead]
If you performed the SVM migration with source cleanup disabled, you can remove the source SVM manually after client cutover is complete.

.Steps

. Verify the status is ready for source cleanup:
+
[source,cli]
----
vserver migrate show
----

. Clean up the source:
+
[source,cli]
----
vserver migrate source-cleanup -vserver <SVM_name>
----
