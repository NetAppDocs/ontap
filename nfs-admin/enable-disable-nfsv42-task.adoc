= Enable or disable NFSv4.2 for ONTAP SVMs
:icons: font
:imagesdir: ../media/

[.lead]
Beginning with ONTAP 9.19.1, you can enable or disable NFSv4.2 by modifying the `-v4.2` option. This allows file access for clients using the NFSv4.2 protocol. 

NFSv4.2 support is managed independently of NFSv4.1 settings, allowing each SVM to advertise or suppress NFSv4.2 capabilities as needed. By default, NFSv4.2 is enabled when a new NFS service is created unless explicitly disabled by the administrator. 

.Step

. Perform one of the following actions:
+
[cols="2*",options="header"]
|===
| If you want to...| Enter the following command...
a|
Enable NFSv4.2
a|
[source,cli]
----
vserver nfs modify -vserver vserver_name -v4.2 enabled
----
a|
Disable NFSv4.2
a|
[source,cli]
----
vserver nfs modify -vserver vserver_name -v4.2 disabled
----
|===
