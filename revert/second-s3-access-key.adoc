= Remove second S3 access keys before reverting from ONTAP 9.19.1

:icons: font
:imagesdir: ../media/

[.lead]
If you upgraded volumes to ONTAP 9.19.1, and added a second S3 access key to any user, these keys must be removed before you can revert to an earlier ONTAP release.

.Steps

. link:regenerate-access-keys.html#show-s3-access-keys[Show S3 access keys] to determine if second access keys have been added for any S3 users that have been upgraded to ONTAP 9.19.1 or later.

. link:regenerate-access-keys.html#remove-an-s3-access-key[Remove the second access keys] for all S3 users.
