= Disable automatic enablement of Autonomous Ransomware Protection before reverting from ONTAP 9.18.1

:icons: font
:imagesdir: ../media/

[.lead]
If you upgraded volumes to ONTAP 9.18.1, ONTAP ARP automatic enablement might have been set for your volumes after a brief grace period (12 hours). It's recommended that you disable this automatic enablement setting on volumes upgraded to ONTAP 9.18.1 before reverting to ONTAP 9.17.1 or earlier.

.Steps

. Determine if the automatic enablement option has been activated on volumes that have been upgraded to ONTAP 9.18.1 or later:
+
[source, cli]
----
security anti-ransomware auto-enable show -vserver <vserver name>
----

. Disable the automatic enablement option for ransomware protection on all volumes on the SVM:
+
[source, cli]
----
security anti-ransomware volume disable -volume * -auto-enabled-volumes-only true -vserver <vserver name>
----
