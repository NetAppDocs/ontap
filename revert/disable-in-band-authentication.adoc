= Disable in-band authentication over NVMe before reverting from ONTAP 9.12.1
:icons: font
:imagesdir: ../media/

[.lead]
If you are running the NVME protocol and you are reverting from ONTAP 9.12.1, you must disable in-band authentication before you revert.  If in-band authentication using DH-HMAC-CHAP is not disabled, revert will fail.
