= Manage and monitor ONTAP SVMs with System Manager
:toclevels: 1
:hardbreaks:
:nofooter:
:icons: font
:linkattrs:
:imagesdir: ./media/

[.lead]
Beginning in ONTAP 9.19.1, cluster administrators can provide their SVM administrators with access to monitor and manage SVMs. SVM administrators can monitor and manage SVMs with the SVM administrator dashboard in ONTAP System Manager.

.Before you begin
Your cluster administrator must provide you with the:

* IP address to the SVM
* Credentials (username and password) for the SVM System Manager dashboard
* Optionally, permissions to add Volumes on the SVM

.Steps

. Enter the management IP address of the SVM in a browser.
. Enter the username and password for the SVM administrator.
. Click *Sign in*. 
+
The ONTAP System Manager dashboard opens.
. View the Protocol details, including the protocols enabled on the SVM. 
. View the Details, including NIS and LDAP information, language and the retention period after a volume is deleted.
. Monitor performance by the Hour, Day, Week, Month, or Year for the selected protocol.
+
You can monitor the Latency, IOPS, and Throughput performance statistics. 
+
You can download the data in a comma separated value (csv) file for further analysis. 
. Select *Storage > Volumes*.
+
The Volumes dashboard page opens.
+
If you have permission to add a Volume, the *Add* button appears.
