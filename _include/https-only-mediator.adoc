* When you install or upgrade to ONTAP Mediator 1.12 or later, you have the option to install ONTAP Mediator with iSCSI and HTTPS support, or support for HTTPS only. If you choose to install ONTAP Mediator with support for HTTPS only, the SCST package is not installed. 

* If you upgrade from a previous version of ONTAP Mediator with SCST installed, and you choose the HTTPS only option during the upgrade, the SCST package is uninstalled and the new ONTAP Mediator version will not contain the SCST package. 

* Installing ONTAP Mediator with support for HTTPS only is supported for the following use cases:

** You plan to only use ONTAP Mediator to monitor your SnapMirror active sync clusters.
** You plan to use an ONTAP Mediator configured with the HTTPS protocol to monitor a single MetroCluster IP configuration.
+
CAUTION: In MetroCluster IP configurations, when ONTAP Mediator is installed with support for HTTPS only, it can only be used to monitor one MetroCluster IP configuration. Monitoring multiple MetroCluster IP configurations is not supported. You must carefully evaluate the requirements for your MetroCluster IP environment before you choose to install ONTAP Mediator with support for HTTPS and not iSCSI. 

* ONTAP Mediator must be installed with support for iSCSI and HTTPS in the following scenarios:
** You plan to configure ONTAP Mediator with iSCSI to monitor a MetroCluster IP configuration.
** You plan to use ONTAP Mediator to monitor more than one MetroCluster IP configurations.
