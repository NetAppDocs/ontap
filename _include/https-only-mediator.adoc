* When you install or upgrade to ONTAP Mediator 1.12 or later, you have the option to install ONTAP Mediator with iSCSI and HTTPS support, or support for HTTPS only. If you choose to install ONTAP Mediator with support for HTTPS only, the SCST package is not installed. 

* If you upgrade from a previous version of ONTAP Mediator with SCST installed, and you choose the HTTPS only option during the upgrade, the SCST package is uninstalled and the new ONTAP Mediator version will not contain the SCST package. 

* Installing ONTAP Mediator with support for HTTPS and not iSCSI is *only* supported for the following use case:

** You plan to use an ONTAP Mediator configured with the HTTPS protocol to monitor a single MetroCluster IP configuration.
+
CAUTION: When installed with support for HTTPS and not iSCSI, ONTAP Mediator can only be used to monitor one MetroCluster IP configuration. Monitoring multiple configurations is not supported. You must carefully evaluate the requirements for your MetroCluster IP environment before you choose to install ONTAP Mediator with support for HTTPS and not iSCSI. 

* ONTAP Mediator must be installed with support for iSCSI and HTTPS for all other configurations.
