= Learn about ONTAP LIFs for SAN environments
:icons: font
:imagesdir: ../media/

[.lead]

LUNs and namespaces in your ONTAP system are contained inside storage virtual machines (SVMs). Network Interfaces (LIFs) allow your SAN clients to connect to an SVM and access data in the LUNs and namespaces it contains.

* You should configure one management LIF for every storage virtual machine (SVM) supporting SAN.
* You can configure two LIFs per node, one for each fabric being used with FC and to separate Ethernet networks for iSCSI.
* Direct connect or the use of Ethernet switches is supported for connectivity. You must configure LIFs for both types of connectivity.
