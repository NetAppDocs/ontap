= Failed to create snapshot
:icons: font
:imagesdir: ../media/

[.lead]
* *Message*
+
`Failed to create snapshot`

* *Cause*
+
The volume already contains the maximum number of snapshots.

* *Corrective action*
+
Delete some snapshots and then retry the backup operation.
