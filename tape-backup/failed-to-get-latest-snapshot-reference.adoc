= Failed to get latest snapshot
:icons: font
:imagesdir: ../media/

[.lead]
* *Message*
+
`Failed to get latest snapshot`

* *Cause*
+
The latest snapshot might not exist because the volume is being initialized by SnapMirror.

* *Corrective action*
+
Retry after initialization is complete.
